## 3.1 RNA 的结构、分类和功能

### 3.1.1 RNA 的结构特点

#### 化学结构
- **核糖**：含有2'-OH基团（与DNA的关键区别）
  - 2'-OH使RNA易被碱水解
  - 使RNA糖环更倾向于C3'-内式构象
- **碱基**：A（腺嘌呤）、U（尿嘧啶）、G（鸟嘌呤）、C（胞嘧啶）
  - U代替T，U与A配对
- **磷酸二酯键**：3'-5'磷酸二酯键连接

#### 二级结构
- **单链特性**：RNA通常是单链，可折叠形成复杂二级结构
- **发夹结构（hairpin）/茎环结构**：
  - 由反向互补序列形成
  - 茎部：Watson-Crick配对
  - 环部：未配对的核苷酸
- **内部环（internal loop）**：茎部两侧均有未配对碱基
- **膨胀环（bulge loop）**：茎部一侧有未配对碱基
- **假结（pseudoknot）**：环序列与茎外序列配对形成的复杂结构

#### 三级结构
- **tRNA**：
  - 二级结构：三叶草结构
    - 氨基酸臂（接受臂）
    - D臂（含二氢尿苷）
    - 反密码子臂
    - TΨC臂（含假尿苷）
  - 三级结构：L型三维结构
- **rRNA**：
  - 复杂的三维折叠
  - 形成核糖体的结构和催化中心

#### RNA与DNA的关键区别

| 特征 | DNA | RNA |
|:----:|:---:|:---:|
| **五碳糖** | 脱氧核糖（2'-H） | 核糖（2'-OH） |
| **碱基** | A、T、G、C | A、U、G、C |
| **结构** | 通常是双螺旋 | 单链，可折叠形成二级/三级结构 |
| **稳定性** | 稳定（双链保护） | 不稳定（易被RNase降解，碱水解） |
| **存在位置** | 主要在细胞核 | 细胞核、细胞质、细胞器 |
| **功能** | 遗传信息储存 | 遗传信息传递、催化、调控等 |

### 3.1.2 RNA 在细胞中的分布

#### 原核生物
- **细胞质**：tRNA、rRNA、mRNA
- 转录和翻译在同一空间进行

#### 真核生物
| 区域 | RNA类型 |
|:----:|:--------|
| **细胞核** | pre-mRNA、snRNA、snoRNA |
| **细胞质** | mRNA、tRNA、rRNA |
| **线粒体** | 自有mt-rRNA、mt-tRNA |
| **叶绿体** | 自有cp-rRNA、cp-tRNA |

### 3.1.3 RNA 的功能分类

| 类型 | 缩写 | 功能 | 含量/特点 |
|:----:|:----:|:----:|:---------|
| 信使RNA | mRNA | 蛋白质合成的模板 | 占总量~5%，种类最多 |
| 转运RNA | tRNA | 携带氨基酸到核糖体 | 占总量~15%，含大量修饰碱基 |
| 核糖体RNA | rRNA | 核糖体组成和催化肽键形成 | 占总量~80% |

| 小核RNA | snRNA | 参与pre-mRNA剪接，剪接体组分 | U1、U2、U4、U5、U6等 |
| 小核仁RNA | snoRNA | 指导rRNA加工和修饰 | 指导甲基化、假尿苷化 |

## 3.2 RNA 转录概述

### 3.2.1 RNA 转录与DNA复制的比较

| 特征 | DNA复制 | RNA转录 |
|:----:|:--------|:--------|
| **模板** | 整个基因组 | 特定基因或区域 |
| **产物** | 子代DNA | RNA（mRNA/tRNA/rRNA等） |
| **酶** | DNA聚合酶（Pol I/III） | RNA聚合酶 |
| **引物** | 需要RNA引物 | **不需要引物** |
| **碱基配对** | A-T, G-C | A-U, T-A, G-C |
| **保真性** | 高（有3'→5'外切酶校对活性） | 较低（无校对活性） |
| **对称性** | 半保留复制，双向 | 不对称转录，单向 |
| **方向性** | 5'→3'双向延伸 | 5'→3'单向延伸 |
| **合成速度** | ~1000 nt/min（原核） | ~40-80 nt/min（原核） |
| **后加工** | 切除引物，连接 | 原核少；真核复杂（加帽、加尾、剪接） |

> **考点**：转录不需要引物，这是与DNA复制的核心区别之一。

---

### 3.2.2 转录机器的主要成分——RNA 聚合酶

#### 原核生物RNA聚合酶（E.coli）

##### 核心酶（Core Enzyme）
- **组成**：α₂ββ'ω
  - **α亚基（rpoA）**：
    - 决定启动子特异性（αCTD可结合UP元件）
    - 参与二聚体形成
  - **β亚基（rpoB）**：
    - 催化中心，形成磷酸二酯键
    - 利福平（rifampicin）结合位点
  - **β'亚基（rpoC）**：
    - 结合DNA模板
    - 含有锌指结构
  - **ω亚基（rpoZ）**：
    - 辅助亚基
    - 参与酶的组装和稳定β'

##### 全酶（Holoenzyme）
- **组成**：核心酶 + σ因子
- **功能**：能识别启动子并起始转录

##### σ因子（Sigma Factor）
| σ因子 | 功能 | 识别的启动子类型 |
|:------:|:------|:----------------|
| σ⁷⁰（RpoD） | 主要σ因子，看家基因 | 标准启动子（-35/-10） |
| σ³²（RpoH） | 热激响应 | 热激基因启动子 |
| σ⁵⁴（RpoN） | 氮代谢 | 氮同化基因启动子 |
| σᴱ（RpoE） | 胞外应激 | 外膜蛋白基因 |
| σᶠ（RpoF） | 鞭毛合成 | 鞭毛基因 |

> **考点**：σ因子负责启动子识别，不同σ因子使RNA聚合酶能够响应不同环境条件转录不同基因。

---

#### 真核生物RNA聚合酶

| 聚合酶 | 定位 | 转录产物 | 相对活性 | 鹅膏蕈碱敏感性 |
|:------:|:----:|:---------|:--------:|:--------------:|
| **Pol I** | 核仁 | 45S pre-rRNA（→18S、5.8S、28S rRNA） | 50-70% | 不敏感 |
| **Pol II** | 核质 | pre-mRNA → mRNA<br>snRNA、miRNA | 20-40% | 低度敏感 |
| **Pol III** | 核质 | tRNA、5S rRNA<br>其他小RNA（U6、7SL等） | ~10% | 高度敏感 |

> **考点**：鹅膏蕈碱（α-amanita）是毒蘑菇的主要毒素，特异性抑制真核RNA聚合酶。

##### RNA Pol II 的特殊结构 - CTD（C-terminal Domain）
- **结构**：RPB1亚基C端含有重复的七肽序列
  - **序列**：Tyr-Ser-Pro-Thr-Ser-Pro-Ser（Y S P T S P S）
  - **重复次数**：酵母26次，哺乳动物52次
- **功能**：
  - **Ser5磷酸化**：转录起始，招募加帽酶
  - **Ser2磷酸化**：转录延伸和终止，招募加尾和剪接因子
- **意义**：CTD作为平台协调转录与RNA加工

---

#### 真核转录辅助因子

1. **通用转录因子（General Transcription Factors, GTF）**
   - TFIIA、TFIIB、TFIID、TFIIE、TFIIF、TFIIH

2. **中介体复合物（Mediator Complex）**
   - 连接转录因子与RNA Pol II
   - 传递调控信号

3. **染色质重塑复合物**
   - SWI/SNF复合物
   - ISWI复合物
   - CHD复合物

4. **组蛋白修饰酶**
   - HAT（组蛋白乙酰转移酶）
   - HDAC（组蛋白去乙酰化酶）
   - HMT（组蛋白甲基转移酶）

### 3.2.3 启动子与转录起始

#### 原核启动子结构

```
          上游元件          -35区            -10区           转录起始位点
             ↓               ↓                ↓                  ↓
         UP元件          TTGACA  -------  TATAAT  -----------  A/G  +1
       (-40~-60)       (共识序列)    间隔17±1bp   (Pribnow框)    (嘌呤)
```

| 元件 | 位置 | 共识序列 | 功能 |
|:----:|:----:|:--------:|:----:|
| **UP元件** | -40~-60 | - | 结合αCTD，增强转录 |
| **-35区** | -35 | TTGACA | σ因子识别和结合 |
| **间隔区** | 15-19 bp | 17 bp最优 | 恰当的空间距离 |
| **-10区** | -10 | TATAAT | DNA解链起始点 |
| **延伸区** | +1 | - | 转录起始位点 |

> **考点**：
> - -10区又称Pribnow框
> - 转录起始位点（+1）通常是嘌呤（A或G）
> - 间隔区长度对启动子活性影响显著

**启动子强度决定因素**：
1. 与共识序列的匹配程度
2. 间隔区长度（17 bp最优）
3. 是否有UP元件
4. 附近是否有其他调控蛋白结合位点

---

#### 真核启动子结构

##### Pol II 启动子

```
        增强子    CAAT框   GC框    TATA框    起始子    +1
          ↓         ↓      ↓       ↓         ↓       ↓
       远端     -90    -80   -30      -1      ATG
       调控元件    |______|      |_____|
                    近端         核心
                  启动子元件    启动子
```

| 元件类型 | 具体元件 | 位置 | 功能 |
|:--------:|:--------:|:----:|:----:|
| **核心启动子** | TATA框 | -25~-30 | TBP识别，确定转录起点 |
| | 起始子（Inr） | -3~+5 | 围绕转录起始点 |
| | 下游启动子元件（DPE） | +28~+32 | 无TATA框时重要 |
| **近端启动子** | CAAT框 | -70~-90 | 结合NF-I等因子 |
| | GC框 | -90~-110 | 结合SP1等因子 |
| **远端调控元件** | 增强子 | 可远距离 | 距离/方向非依赖激活 |
| | 沉默子 | 可远距离 | 抑制转录 |
| | 绝缘子 | 可远距离 | 阻止增强子/异染色质扩散 |

> **考点**：
> - TATA框是核心启动子的关键元件
> - 增强子的特点是距离和方向非依赖性
> - 某些基因（管家基因）缺少TATA框

##### Pol III 启动子类型

| 类型 | 基因 | 启动子位置 | 控制元件 |
|:----:|:----:|:----------:|:--------:|
| I型 | 5S rRNA | 基因内部 | A框、C框 |
| II型 | tRNA | 基因内部 | A框、B框 |
| III型 | U6 snRNA、7SL RNA | 基因上游 | TATA框、PSE |

## 3.3 RNA转录的基本过程

### 3.3.1 模板识别

#### 原核生物
- σ⁷⁰因子识别-35区（TTGACA）和-10区（TATAAT）
- 全酶与启动子结合形成封闭复合物

#### 真核生物
1. **TFIID**（含TBP）识别并结合TATA框
2. **TFIIA**和**TFIIB**相继结合
3. **TFIIF**协助**RNA Pol II**结合
4. **TFIIE**和**TFIIH**加入，形成完整起始复合物

> **考点**：TBP（TATA-binding protein）是TFIID的核心亚基，负责识别TATA框。

---

### 3.3.2 转录起始

#### 起始阶段

| 步骤 | 名称 | 描述 |
|:----:|:----:|:----:|
| 1 | **封闭复合物形成** | RNA聚合酶结合双链DNA |
| 2 | **开放复合物形成** | DNA解链约17 bp，形成转录泡 |
| 3 | **起始复合物形成** | 合成第一个磷酸二酯键 |
| 4 | **启动子清除** | RNA聚合酶离开启动子，σ因子脱离（原核） |

#### 流产起始（Abortive Initiation）
- **现象**：RNA聚合酶合成短RNA（2-9 nt）后释放
- **机制**：RNA-DNA杂交不稳定，酶在启动子附近滑动
- **意义**：重复多次直到成功延伸，确保转录准确性

> **考点**：转录起始不需要引物，RNA聚合酶可以直接起始RNA合成。

---

### 3.3.3 转录延伸

#### 转录泡结构
```
        上游DNA      转录泡（~17bp）      下游DNA
            ↓            ↓                 ↓
    ========    ==========    ========
           |    RNA-DNA杂交 |  |
           |     （~8-9 bp）  |  |
           ↓                  ↓
         新生RNA（5'→3'）
```

#### 延伸特点
- **方向**：5'→3'
- **模板链**：反义链（antisense strand）
- **编码链**：有义链（sense strand），与RNA序列相同（T→U）

#### 延伸因子

| 生物 | 延伸因子 | 功能 |
|:----:|:--------:|:----:|
| 原核 | NusA | 促进暂停，辅助终止 |
| | NusG | 促进延伸，与核糖体偶联 |
| 真核 | TFIIS | 刺激内切酶活性，拯救回退的聚合酶 |

#### Pausing和Backtracking
- **暂停（Pausing）**：延伸过程中暂时停止
  - 原因：DNA序列、二级结构、调控因子
- **回退（Backtracking）**：RNA聚合酶向后移动，RNA 3'端从活性位点移出
  - 需要：TFIIS（真核）或GreA/GreB（原核）来拯救

---

### 3.3.4 转录终止

#### 原核生物终止机制

##### 1. 不依赖ρ因子的终止（Intrinsic Termination）

**特征序列**：
```
      GC富含区（形成发夹）      U尾
          ↓                      ↓
    CGGCG-----CCGC    AAAAAA...
        ||||||         ||||||
    GCCGC------------GGCGU
```

**机制**：
1. GC富含区形成稳定的发夹结构
2. 发夹导致RNA聚合酶暂停
3. 后面的poly-U序列与DNA形成的A-U杂交不稳定
4. RNA-DNA杂交解离，转录终止

**条件**：
- 发夹结构（茎部通常GC富含）
- 发夹后紧跟poly-U（通常≥6个U）

##### 2. 依赖ρ因子的终止（Rho-dependent Termination）

**ρ因子特性**：
- 六聚体环状结构
- 具有ATP酶和解旋酶活性

**终止过程**：
1. ρ因子识别rut位点（Rho utilization site）
   - C富含、G贫乏
   - 无二级结构
2. ρ因子沿5'→3'方向追赶RNA聚合酶
3. 到达转录泡后，利用ATP水解的能量解离RNA-DNA杂交
4. 释放RNA产物

> **考点**：不依赖ρ因子的终止需要发夹结构+poly-U；依赖ρ因子的终止需要rut位点。

---

#### 真核生物终止机制

| 聚合酶 | 终止机制 |
|:------:|:--------:|
| **Pol I** | 由终止因子（Reb1等）识别特定序列 |
| **Pol II** | 与切割和多聚腺苷化相关<br>AAUAAA信号下游切割， torpedo模型 |
| **Pol III** | poly-T终止信号（类似原核） |

##### Pol II 终止模型

**Torpedo模型**：
1. CPSF/CstF识别AAUAAA并切割pre-mRNA
2. XRN2（5'→3'外切酶）降解下游RNA
3. XRN2追赶Pol II并导致其终止

**Allosteric模型**：
- RNA加工导致Pol II构象变化，降低延伸效率

## 3.4 原核生物与真核生物的转录及产物特征比较

### 3.4.1 转录过程比较

| 特征 | 原核生物 | 真核生物 |
|:----:|:--------:|:--------:|
| **RNA聚合酶** | 1种（多亚基） | 3种（Pol I/II/III） |
| **启动子结构** | -35区/-10区 | TATA框、起始子等 |
| **转录因子** | σ因子 | 多种通用转录因子 |
| **转录与翻译** | 偶联（同时进行） | 分隔（核内转录，质内翻译） |
| **转录后加工** | 极少 | 复杂（5'帽、3'尾、剪接） |
| **终止机制** | ρ依赖/不依赖 | 与加尾信号关联 |
| **mRNA结构** | 多顺反子 | 单顺反子 |

---

### 3.4.2 原核生物 mRNA 的特征

| 特征 | 描述 |
|:----:|:----:|
| **多顺反子（Polycistronic）** | 多个基因在同一mRNA上，从同一启动子转录 |
| **无5'帽子** | 直接从5'-三磷酸起始 |
| **无3' poly-A尾** | 无poly-A尾巴 |
| **无内含子** | 原核基因一般无内含子，无需剪接 |
| **SD序列（Shine-Dalgarno）** | AGGAGG，位于起始密码子上游3-10 bp<br>核糖体结合位点，与16S rRNA 3'端互补 |
| **半衰期短** | 约2-5分钟，快速降解 |
| **5'UTR和3'UTR** | 存在调控元件 |

> **考点**：SD序列是原核mRNA的核糖体结合位点，与16S rRNA互补配对。

---

### 3.4.3 真核生物 mRNA 的特征

| 特征 | 描述 |
|:----:|:----:|
| **单顺反子（Monocistronic）** | 通常一个基因产生一个mRNA |
| **5'帽子结构** | m⁷GpppN（7-甲基鸟苷帽子）<br>功能：保护mRNA、促进翻译、核输出 |
| **3' poly-A尾巴** | 约200个A残基<br>功能：保护mRNA、促进翻译、核输出 |
| **内含子/外显子** | 需要剪接去除内含子 |
| **5'UTR和3'UTR** | 含有丰富调控元件 |
| **半衰期较长** | 数小时到数天 |
| **Kozak序列** | GCCRCCAUGG（R=嘌呤）<br>起始密码子周围，增强翻译效率 |

#### 5'帽子结构（5' Cap）

| 类型 | 结构 |
|:----:|:----:|
| **Cap-0** | m⁷GpppN |
| **Cap-1** | m⁷GpppNᵐ（第一个核糖2'-O甲基化） |
| **Cap-2** | m⁷GpppᵐNᵐ（前两个核糖2'-O甲基化） |

> **考点**：5'帽子是m⁷G通过5'-5'三磷酸键连接的。

#### Poly-A尾巴的功能
1. 保护mRNA免受3'外切酶降解
2. 促进翻译起始（与PABP/eIF4G相互作用）
3. 参与核输出
4. 标记成熟mRNA（与组蛋白mRNA区别）

> **考点**：组蛋白mRNA无poly-A尾巴，其3'端形成茎环结构。

## 3.5 原核生物 RNA 聚合酶与 RNA 转录

### 3.5.1 原核生物 RNA 聚合酶（补充）

#### 基因与亚基对应关系

| 基因 | 亚基 | 功能 |
|:----:|:----:|:----:|
| rpoA | α | 启动子识别，二聚体形成 |
| rpoB | β | 催化中心，利福平结合 |
| rpoC | β' | DNA结合 |
| rpoZ | ω | 组装和稳定 |

#### σ因子的等级调控

```
          σ⁷⁰（主要）
             |
        ┌────┴────┐
     σ³²      σ⁵⁴
    (热激)    (氮代谢)
        |
        σᴱ      σᶠ
     (胞外应激)  (鞭毛)
```

> **考点**：不同σ因子使细菌能够快速响应环境变化，转录不同基因组合。

---

### 3.5.2 原核生物启动子结构（补充）

#### 共识序列

| 元件 | 共识序列 | 保守性 |
|:----:|:--------:|:------:|
| -35区 | TTGACA | 高度保守 |
| -10区 | TATAAT | 高度保守 |
| 间隔区 | 17±1 bp | 长度比序列更重要 |

#### 启动子突变效应

| 突变类型 | 效应 |
|:--------:|:----:|
| **Down mutation** | 与共识序列差异增大，降低启动子活性 |
| **Up mutation** | 更接近共识序列，提高启动子活性 |
| **间隔区突变** | 改变长度（非17 bp）显著降低活性 |

## 3.6 真核生物 RNA 聚合酶与 RNA 转录

### 3.6.1 真核生物 RNA 聚合酶

#### Pol I（核仁）
- **亚基组成**：14个亚基
- **功能**：转录45S pre-rRNA
- **加工**：45S → 18S + 5.8S + 28S rRNA

#### Pol II（核质）
- **亚基组成**：12个亚基
- **特殊结构**：CTD（C-terminal domain）
- **功能**：
  - 转录蛋白质编码基因（mRNA）
  - 转录大部分snRNA
  - 转录部分miRNA

#### Pol III（核质）
- **亚基组成**：17个亚基
- **功能**：
  - 转录tRNA
  - 转录5S rRNA
  - 转录U6 snRNA、7SL RNA、其他小RNA

---

### 3.6.2 真核生物启动子结构

#### Pol II 启动子类型

| 类型 | 特征 | 基因举例 |
|:----:|:----:|:--------:|
| **富含TATA框** | 有明确TATA框 | 发育调控基因 |
| **无TATA框** | 无TATA框，有Inr/DPE | 管家基因 |
| **GC富含** | 多个GC框 | 持续表达基因 |

---

### 3.6.3 真核转录因子

#### 通用转录因子（GTF）

| 因子 | 主要功能 | 关键亚基/特点 |
|:----:|:--------:|:-------------:|
| **TFIID** | 识别TATA框，组装起始复合物 | TBP + TAFs（TBP相关因子） |
| **TFIIA** | 稳定TFIID-DNA结合 | - |
| **TFIIB** | 招募Pol II，确定转录方向 | 结合TBP和Pol II |
| **TFIIF** | 结合Pol II，抑制非特异性结合 | RAP30/74 |
| **TFIIE** | 激活TFIIH | α/β亚基 |
| **TFIIH** | 解旋酶活性（XPB/XPD）<br>激酶活性（CTD磷酸化） | 10个亚基 |

> **考点**：TFIIH具有解旋酶和激酶双重活性，是转录起始必需的。

#### 转录因子组装顺序（预起始复合物PIC）

```
1. TFIID 结合 TATA框
   ↓
2. TFIIA 和 TFIIB 结合
   ↓
3. TFIIF-Pol II 复合物结合
   ↓
4. TFIIE 结合
   ↓
5. TFIIH 结合 → 完整PIC形成
   ↓
6. DNA解链，转录起始
```

## 3.7 真核生物 RNA 的转录后加工

### 3.7.1 5'端加帽（5' Capping）

#### 加帽时机
- 转录开始后20-30 nt

#### 加帽酶

| 酶 | 反应 |
|:--:|:----:|
| **RNA三磷酸酶** | 去除5'端γ-磷酸 |
| **鸟苷酰转移酶** | 添加GMP（5'-5'三磷酸键） |
| **甲基转移酶** | 添加7-甲基鸟苷（m⁷G） |

#### 加帽过程
```
原始转录本：
pppN...

步骤1（三磷酸酶）：
ppN...

步骤2（鸟苷酰转移酶）：
GpppN...

步骤3（甲基转移酶）：
m⁷GpppN... (Cap-0)
```

#### 加帽功能
1. 保护mRNA免受5'外切酶降解
2. 促进mRNA核输出
3. 被翻译起始因子eIF4E识别，启动翻译
4. 参与剪接

---

### 3.7.2 3'端加尾（3' Polyadenylation）

#### 加尾信号
- **加尾信号**：AAUAAA（高度保守）
- **下游元件**：UGUA或GU丰富区
- **切割位点**：AAUAAA下游10-30 nt

#### 加尾复合物

| 复合物 | 功能 |
|:------:|:----:|
| **CPSF**（切割和多聚腺苷化特异性因子） | 识别AAUAAA |
| **CstF**（切割刺激因子） | 结合下游GU丰富区 |
| **CFI/CFII**（切割因子I/II） | 辅助切割 |
| **PAP**（poly-A聚合酶） | 添加poly-A尾巴 |
| **PABP**（poly-A结合蛋白） | 结合poly-A尾 |

#### 加尾过程
1. CPSF识别AAUAAA
2. CstF结合下游元件
3. CFI/CFII加入
4. 在切割位点切割pre-mRNA
5. PAP添加poly-A尾巴（约200个A）
6. PABP结合并调控尾巴长度

---

### 3.7.3 剪接（Splicing）

#### 剪接体组成
- **snRNP**：snRNA + 蛋白质
  - U1 snRNP
  - U2 snRNP
  - U4 snRNP
  - U5 snRNP
  - U6 snRNP

#### 剪接信号

| 元件 | 位置 | 共识序列 |
|:----:|:----:|:--------:|
| **5'剪接位点** | 内含子5'端 | GU |
| **分支点** | 内含子内部 | CURAY（R=嘌呤，Y=嘧啶） |
| **3'剪接位点** | 内含子3'端 | AG |

> **考点**：GU-AG法则（GT-AG法则）是剪接位点的基本规则。

#### 剪接反应（两步转酯反应）

**第一步**：
- 分支点A的2'-OH攻击5'剪接位点的磷酸二酯键
- 形成5'-2'磷酸二酯键，产生套索（lariat）结构

**第二步**：
- 外显子1的3'-OH攻击3'剪接位点
- 外显子连接，释放内含子套索

```
外显子1 - 内含子 - 外显子2
           ↓
外显子1 外显子2（成熟mRNA）
           ↓
内含子套索 → 降解
```

#### 剪接体组装顺序
```
1. U1 结合5'剪接位点
2. U2 辅助因子（U2AF）结合3'剪接位点
3. U2 结合分支点
4. U4/U6•U5 三聚体加入
5. 结构重排，激活剪接体
```

#### 可变剪接（Alternative Splicing）
- **定义**：一个基因产生多种mRNA
- **比例**：>95%的人类多外显子基因发生可变剪接
- **类型**：
  | 类型 | 描述 |
  |:----:|:----:|
  | 外显子跳过 | 最常见类型 |
  | 内含子保留 | 在某些基因中 |
  | 互斥外显子 | 只选一个外显子 |
  | 可变5'剪接位点 | 不同的5'位点 |
  | 可变3'剪接位点 | 不同的3'位点 |

> **考点**：可变剪接是蛋白质组多样性的重要来源。

## 3.8 转录调控（进阶）

### 3.8.1 染色质水平调控

#### 染色质结构
- **常染色质**：疏松，活跃转录
- **异染色质**：致密，转录沉默

#### 组蛋白修饰

| 修饰类型 | 效应 | 酶 |
|:--------:|:----:|:---:|
| 乙酰化 | 激活转录 | HAT（乙酰转移酶）<br>HDAC（去乙酰化酶） |
| 甲基化 | 激活或抑制 | HMT（甲基转移酶）<br>HDM（去甲基化酶） |
| 磷酸化 | 多种效应 | 激酶/磷酸酶 |
| 泛素化 | 信号转导 | E3泛素连接酶 |

> **考点**：组蛋白乙酰化通常激活转录；组蛋白甲基化的效应取决于位点。

#### DNA甲基化
- **位置**：CpG二核苷酸（CpG岛）
- **效应**：通常抑制转录
- **机制**：阻止转录因子结合，招募抑制性复合物

#### 染色质重塑
- **SWI/SNF复合物**：利用ATP滑动/驱逐核小体
- **ISWI复合物**：规则核小体间距
- **CHD复合物**：染色质修饰和重塑

---

### 3.8.2 表观遗传调控

#### 组蛋白密码假说
- 组蛋白修饰的组合决定染色质状态
- 被"读取器"（Reader）蛋白识别

#### X染色体失活
- **Xist RNA**：包裹失活X染色体
- **剂量补偿**：雌性哺乳细胞两条X染色体一条失活

#### 基因组印记
- 亲本来源特异性的基因表达
- DNA甲基化是主要机制

## 考研重点总结

### 核心概念
1. **转录不需要引物**（与DNA复制关键区别）
2. **不对称转录**：只有一条DNA链作为模板
3. **σ因子**：原核启动子识别的关键因子
4. **TBP**：真核TATA框识别的关键蛋白
5. **CTD磷酸化**：协调转录与RNA加工
6. **GU-AG法则**：剪接位点识别规则
7. **可变剪接**：蛋白质组多样性的重要来源

### 常考对比
| 对比项 | 原核 | 真核 |
|:------:|:----:|:----:|
| RNA聚合酶 | 1种 | 3种 |
| 启动子 | -35/-10 | TATA框 |
| 转录因子 | σ因子 | 多种GTF |
| mRNA | 多顺反子 | 单顺反子 |
| 5'帽 | 无 | 有 |
| 3' poly-A | 无 | 有 |
| 内含子 | 无 | 有 |
| 转录-翻译 | 偶联 | 分隔 |

### 关键序列
| 序列 | 功能 |
|:----:|:----:|
| TTGACA | 原-35区 |
| TATAAT | 原-10区（Pribnow框） |
| TATAAA | 真TATA框 |
| AAUAAA | 加尾信号 |
| AGGAGG | SD序列 |
| GU-AG | 剪接位点 |
| GCCRCCAUGG | Kozak序列 |
